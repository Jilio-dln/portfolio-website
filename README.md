# Portfolio Website

Dit is de repository voor mijn portfolio website die ik ontwikkel binnen het OPO **Websites** van de opleiding **Graduaat Programmeren**.

De website dient als een online portfolio waarin ik mijn vaardigheden, projecten en interesses toon. Het doel van deze website is om een overzicht te geven van mijn groei als student developer en de technieken die ik leer toepassen.

## Inhoud van de website

De website bestaat momenteel uit verschillende pagina’s:

- **Home (index.html)** – introductie en overzicht van mijn portfolio
- **CV** – informatie over mijn opleiding, werkervaring en vaardigheden
- **Projecten** – een overzicht van projecten en oefeningen uit mijn opleiding
- **Interesses** – mijn hobby’s en persoonlijke interesses
- **Contact (contact.html)** – manieren waarop een werkgever mij kan bereiken

## Gebruikte technologieën

Voor het ontwikkelen van deze website gebruik ik:

- **HTML** – structuur van de webpagina’s  
- **CSS** – styling en layout van de website  

## Doel van dit project

Dit project helpt mij om:

- correcte HTML-structuren op te bouwen  
- webpagina’s vorm te geven met CSS  
- een consistente layout en navigatie te creëren  
- mijn werk en projecten online te presenteren
